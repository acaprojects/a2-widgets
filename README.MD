# Angular 2 Widgets

## Usage

Import the desired widget into your component.

### Button Toggle

Import `import { ButtonToggle } from '@aca/a2-widgets';`

### Button Groups

Import `import { ButtonGroup } from '@aca/a2-widgets';`

### Toggle Buttons

Import `import { FancyToggle } from '@aca/a2-widgets';`
